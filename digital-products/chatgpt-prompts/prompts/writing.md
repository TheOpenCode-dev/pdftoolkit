# Writing Prompts (40)

## Blog Posts
```
Write a comprehensive blog post about [TOPIC]. Include:
- A compelling introduction
- 3-5 main sections with subheadings
- Practical tips readers can apply
- A strong conclusion with CTA

Target audience: [DESCRIPTION]
Word count: [NUMBER]
Tone: [PROFESSIONAL/FRIENDLY/CASUAL]
```

```
Create a listicle of [NUMBER] [TOPIC]. Each item should have:
- A catchy title
- 2-3 sentences of explanation
- One actionable tip

Write in a [TONE] tone.
```

## Emails
```
Write a professional cold email to [TARGET] about [PRODUCT/SERVICE].

```

```
Write a follow-up email for [SITUATION]. Make it:
```

## Essays
```

1. Introduction with thesis
2. Body paragraph 1 - main argument
3. Body paragraph 2 - supporting evidence
4. Body paragraph 3 - counterargument & rebuttal
5. Conclusion

Use formal academic tone.
```
