# Python Automation Scripts Pack

Boost your productivity with these 10 ready-to-use Python scripts.

## What's Included

1. **File Organizer** - Automatically sorts files in a folder by type (images, docs, videos, etc.)
2. **Image Resizer** - Batch resize images to any dimension
3. **CSV Cleaner** - Remove duplicates, fix encoding, merge CSVs
4. **Screenshot Tool** - Take screenshots from command line (cross-platform)
5. **URL Extractor** - Extract all URLs from text/HTML files
6. **Folder Sync** - Mirror folder contents with rsync-like behavior
7. **PDF Page Counter** - Count pages in multiple PDFs
8. **Text to Speech** - Convert text files to audio
9. **Duplicate Finder** - Find duplicate files by hash
10. **Batch Renamer** - Rename multiple files with patterns

## Requirements

- Python 3.8+
- See `requirements.txt` for dependencies

## Usage

Each script is standalone. Run with:
```bash
python script_name.py
```
